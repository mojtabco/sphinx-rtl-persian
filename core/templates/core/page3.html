4.1.3. لیست های گزینه

برای فهرست کردن گزینه های خط فرمان:

-a

گزینه خط فرمان "a"

-b file

گزینه ها می توانند استدلال و توضیحات طولانی داشته باشند

--long

گزینه ها نیز می توانند طولانی باشند

--input=file

گزینه های طولانی نیز می توانند استدلال داشته باشند

--very-long-option

توضیحات همچنین می تواند از خط بعدی شروع شود.

توضیحات ممکن است شامل چندین عنصر بدنه باشد، صرف نظر از اینکه از کجا شروع می شود.

-x, -y, -z

گزینه های متعدد یک "گروه گزینه" هستند.

-v, --verbose

معمولا دیده می شود: گزینه های کوتاه و بلند.

-1 file, --one=file, --two file

گزینه های متعدد با آرگومان

/V

گزینه های به سبک DOS/VMS نیز

بین گزینه و توضیحات باید حداقل دو فاصله وجود داشته باشد.
